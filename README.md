
# Wano editor

Wano is a web-based alternative to Nano.

## Requirements
- pyngrok
- uuid
- flask

## Installation
`git clone https://github.com/lucydjo/wano`

`cd wano`

`python setup.py install`

## Ngrok configuration
Add your Ngrok auth key in env

`export NGROK_WANO_KEY=your_ngrok_auth_key`

## Usage
`wano anyfile.txt`
